# Barbearia+ — Sistema de Agendamento Online

## 📌 Sobre o Projeto

O **Barbearia+** é um sistema de agendamento online desenvolvido como MVP (Minimum Viable Product) utilizando conceitos de:

* Lean Inception
* Scrum
* Engenharia de Requisitos
* Gestão Ágil de Produtos
* Gestão Ágil de Projetos
* Design e Prototipação de Interface

O objetivo principal do sistema é permitir que clientes realizem agendamentos online de maneira simples, organizada e eficiente, melhorando a experiência do usuário e otimizando o gerenciamento da barbearia.

---

# 🎯 Objetivo do MVP

O MVP foi desenvolvido para validar a seguinte hipótese:

> Clientes de barbearias possuem dificuldade em realizar agendamentos de forma prática e organizada utilizando meios tradicionais, como mensagens e ligações.

Com isso, o sistema busca:

* reduzir conflitos de horários;
* facilitar o gerenciamento da agenda;
* melhorar a experiência do cliente;
* digitalizar o processo de atendimento.

# 📌 MVP Canvas

O MVP foi estruturado visando validar as funcionalidades essenciais relacionadas ao processo de agendamento online.

* Cadastro de usuários
* Login

* Agendamento de horários

# 📌 Requisitos Não Funcionais

* O sistema deve responder operações principais em até 2 segundos.
* O sistema deve ser responsivo para dispositivos móveis.
* As senhas dos usuários devem ser armazenadas de forma criptografada.
* O sistema deve seguir boas práticas relacionadas à LGPD.

---

# ✅ Definition of Ready

Uma história estará pronta para desenvolvimento quando:

* possuir descrição clara;
* possuir critérios de aceitação;
* possuir prioridade definida;
* possuir estimativa em story points;
* possuir dependências identificadas;
* possuir entendimento compartilhado pelo time.

---

# ✅ Definition of Done

Uma história será considerada concluída quando:

* critérios de aceitação forem atendidos;
* testes executados com sucesso;
* código revisado;
* funcionalidade integrada;
* sem bugs críticos;
* responsividade validada;
* tempo de resposta inferior a 2 segundos;
* dados protegidos conforme LGPD.
